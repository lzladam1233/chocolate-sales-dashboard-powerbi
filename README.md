# chocolate-sales-dashboard-powerbi
Power BI dashboard analyzing chocolate sales trends, discounts, and profit insights.

## 📊 Overview
This project presents a Power BI dashboard analyzing chocolate sales performance, focusing on revenue trends, discount impact, and profit insights.

## 🛠 Tools Used
- Power BI
- Data Visualization
- DAX (for calculated measures)

## 📸 Dashboard Preview
Refer to Power Bi.pnp file. Thanks!

## 📈 Key Insights
- Higher discount levels are associated with lower profit per transaction, indicating margin erosion
- Revenue remains relatively stable (~2.1M monthly), suggesting discounts are not a primary driver of revenue
- Profit margin is consistently around 40%
- Q3 shows the strongest performance across all quarters
- Customers continue to purchase even with minimal discounts, indicating pricing power

## 💡 Business Recommendations
- Reduce excessive discounting to improve profitability
- Focus on maintaining stable pricing rather than relying on discounts
- Leverage strong-performing periods (Q3) for strategic promotions
